# followup-saas
FollowUp SaaS — A lightweight WhatsApp-based customer retention system for local businesses. Built and deployed for a real salon client (Hair Court Salon, Noida). Replaces manual follow-ups with an automated pipeline: QR registration → Google Sheets → owner dashboard → one-click WhatsApp outreach.

FollowUp is not a CRM. It's the missing last step — the follow-up that never happens.

Built for and validated with a real salon client in Noida. The goal was one outcome: get lost customers to come back. Everything else was cut.

Simplicity was a deliberate choice. The owner needed zero training. If it required explanation, it was too complex.
